# IGNITE 2026 - Inter College Fest Portal 🔥  IGNITE 2026 is a futuristic inter-college fest portal designed for technical, cultural, gaming, and literary events. The platform allows students from different colleges to register for events, apply for accommodation, and generate digital QR-based entry ID cards.  ## Features  * ✨ Modern neon-glow UI with glassmorphism design * 🎭 Multiple event arenas with category-wise navigation * 📝 Basic registration system with auto-filled event forms * 👥 Individual & group participation support * 🏨 Stay portal with dynamic cost calculation * 🎫 Digital ticket-style ID card generation * 📱 QR code-based fest entry pass * 🌌 Animated gradients, hover effects, and interactive UI  ## Tech Stack  * HTML5 * CSS3 * JavaScript * QRCode.js  ## Purpose  This project was built to create a complete and visually immersive fest management portal experience for inter-college events.  Because apparently college fests now require websites cooler than some startup landing pages 💀
